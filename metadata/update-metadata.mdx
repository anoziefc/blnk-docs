## Overview

With the [Update Metadata](/reference/update-metadata) endpoint, you can modify existing metadata or add new metadata to an item after it has been created.

## How it works

<Steps titleSize="h3">
  <Step title="Initial metadata">

    Consider the following metadata attached to an item:

    ```json
    {
      ...
      "meta_data": {
        "first_name": "Jason"
      }
    }
    ```
  </Step>
  <Step title="Adding more metadata">
    To add a last name and verification status, send a POST request to:

    ```
    POST YOUR_BLNK_INSTANCE_URL/{id}/metadata
    ```

    ```json Request
    {
      "meta_data": {
        "last_name": "Brooks",
        "verification_status": "Pending"
      }
    }
    ```
  </Step>
  <Step title="Updating metadata">
    To update the verification status, send the following request:

    ```
    POST YOUR_BLNK_INSTANCE_URL/{id}/metadata
    ```

    ```json Request
    {
      "meta_data": {
        "verification_status": "Approved"
      }
    }
    ```
  </Step>
  <Step title="Final metadata">
    After these updates, the metadata for the item will look like this:

    <CodeGroup>
    ```json Before
    {
      ...
      "meta_data": {
        "first_name": "Jason"
      }
    }
    ```
    ```json After
    {
      ...
      "meta_data": {
        "first_name": "Jason",
        "last_name": "Brooks",
        "verification_status": "Approved"
      }
    }
    ```
    </CodeGroup>
  </Step>
</Steps>

Blnk offers a feature that allows you to schedule transactions for future execution with ease. By including the "scheduled_for" parameter in your transaction payload, you can precisely specify when you want a transaction to take place. Here's a breakdown of how it works:

## Schedule a Credit Transaction
```json record transaction
{
    "amount": 200,
    "description": "Commission Earned",
    "reference": "ref-12346",
    "drcr": "Credit",
    "currency": "NGN",
    "balance_id": "bln_0be360ca-86fe-457d-be43-daa3f966d8f0",
    "scheduled_for": "2024-10-08T19:48:00",
    "risk_tolerance_threshold": 0
}
```

```json response
{
    "id": "txn_844c23ef-6cca-45b6-9878-0189e15334f9",
    "reference": "ref-12346",
    "amount": 200,
    "currency": "NGN",
    "description": "Commission Earned",
    "drcr": "Cebit",
    "status": "SCHEDULED",
    "ledger_id": "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc",
    "balance_id": "bln_0be360ca-86fe-457d-be43-daa3f966d8f0",
    "credit_balance_before": 0,
    "debit_balance_before": 0,
    "credit_balance_after": 200,
    "debit_balance_after": 0,
    "balance_before": 0,
    "balance_after": 200,
    "created_at": "2024-02-20T05:37:35.535621126Z",
    "scheduled_for": "2024-10-08T19:48:00",
    "risk_tolerance_threshold": 0,
    "risk_score": 0.0009
}
```

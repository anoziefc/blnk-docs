### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Path
<ParamField path="balance_id" type="string">
  Specifies the unique ID of the balance being retrieved. Leave empty to view all balances in your Blnk Ledger.
</ParamField>

<RequestExample>
```json
curl --request GET \
  --url http://localhost:5001/balances/{balance_id} \
  --header 'Authorization: Basic <encoded-value>'
```
</RequestExample>

### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being retrieved
</ParamField>

### Body
<ParamField body="fields" type="string[]">
  Specifies an array of fields to be tokenized.
</ParamField>

<RequestExample>
```bash
curl --request POST \
  --url http://localhost:5001/identities/{identity_id}/tokenize \
  --header 'Authorization: Basic <encoded-value>' \
  --header 'Content-Type: application/json' \
  --data '{
  "fields": [
    "firstName", 
    "lastName", 
    "emailAddress", 
    "phoneNumber"
  ]
}'
```
</RequestExample>
<ResponseExample>
```json 201
{
    "message": "Fields tokenized successfully"
}
```
</ResponseExample>

### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Body
<ParamField body="name" type="string" required>
  Name of the hook. Ensure it is as descriptive as possible for easy reference.
</ParamField>
<ParamField body="url" type="string" required>
  Endpoint URL where the hook will send requests.
</ParamField>
<ParamField body="type" type="string" required>
  Type of the hook — `PRE_TRANSACTION` or `POST_TRANSACTION`.
</ParamField>
<ParamField body="active" type="boolean" required>
  Indicates whether the hook is enabled or disabled.
</ParamField>
<ParamField body="timeout" type="number" required>
  Maximum time (in seconds) allowed for the hook to execute.
</ParamField>
<ParamField body="retry_count" type="number" required>
  Maximum number of retry attempts if the hook fails.
</ParamField>

<RequestExample>
```json
curl --request POST \
  --url http://localhost:5001/ledgers \
  --header 'Authorization: Basic <encoded-value>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Pre-transaction validation",
    "url": "https://api.example.com/validate",
    "type": "PRE_TRANSACTION",
    "active": true,
    "timeout": 30,
    "retry_count": 3
  }'
```
</RequestExample>

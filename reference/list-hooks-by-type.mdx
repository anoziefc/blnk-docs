### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Body
<ParamField query="type" type="string" required>
  Type of the hook — `PRE_TRANSACTION` or `POST_TRANSACTION`

  ```
  GET http://YOUR_BLNK_INSTANCE_URL/hooks?type=PRE_TRANSACTION
  ```
</ParamField>

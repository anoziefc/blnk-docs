---
title: 'Examples'
description: 'Learn how to deploy blnk server'
---

<Info>
  **Prerequisite**
    - Docker 20.10.11+
    - Compose 1.29.2+
</Info>

Step 1. Clone Blnk to your local server to computer


```bash sh
    git clone https://github.com/jerry-enebeli/Blnk && cd Blnk
```

Step 2. Create a `blnk.json` configuration file.
```bash sh
touch blnk.json
```


Step 3. Copy and paste the below json into your `blnk.json` configuration file.

```bash sh
{
  "project_name": "Blnk",
  "time_zone": "",
  "data_source": {
    "dns": "postgres://postgres:password@postgres:5432/blnk?sslmode=disable"
  },
  "redis": {
    "dns": "redis:6379"
  },
  "account_number_generation": {
    "enable_auto_generation": true,
    "http_service": {
      "url": "http://localhost:5001/mocked-account",
      "headers": {
        "Authorization": "Bearer some_auth_token"
      }
    }
  },
  "server": {
    "domain": "blnk.io",
    "ssl": false,
    "ssl_email": "jerryenebeli@gmail.com",
    "port": "5001"
  },
  "notification": {
    "slack": {
      "webhook_url": "https://hooks.slack.com"
    },
    "webhook": {
      "url": "http://server:5001/webhooks",
      "headers": {}
    }
  }
}
```

Step 4. Start Docker Compose.

```bash docker
docker compose up
```


Blnk server should be running on `http://localhost:5001`.
